# Furniture Center Tech

A professional company website and digital services project with a premium black and gold theme.

## Services
- **App Development**: High-performance mobile applications for Android and iOS.
- **Web Development**: Responsive and fast websites.
- **UI/UX Design**: Elegant and intuitive user interfaces.

## Tech Stack
- **Web**: React, HTML5, CSS3
- **Mobile**: Flutter, Jetpack Compose

## Design Theme
- **Primary Color**: Gold (#D4AF37)
- **Background**: Black (#0A0A0A)
- **Typography**: Playfair Display (Headers), Poppins (Body)

## Project Structure
- `index.html`: Main landing page.
- `style.css`: Custom styling.
- `/src`: Application source code.
